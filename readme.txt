== Description ==

Do SEO for the post, pages and custom post type content images for the Gutenberg images.

See more at <a href="https://maheshwaghmare.com/doc/image-seo/">getting started guide</a>

Extend the plugin from <a href="https://github.com/maheshwaghmare/image-seo/">Github</a>.

== Installation ==

1. Install the <code>Image SEO</code> plugin either via the WordPress plugin directory, or by uploading the files to your server at <code>wp-content/plugins</code>.
2. Activate plugin.
3. Open Page/Post and search for "Image SEO" block
